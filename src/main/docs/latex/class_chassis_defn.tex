\doxysubsection{Detailed Description}
Create a chassis from an X\+ML definition. C\+H\+A\+S\+S\+I\+S\+\_\+\+T\+Y\+PE (\mbox{\hyperlink{_chassis_factory_8h}{Chassis\+Factory.\+h}}) determines the type of chassis to create.
